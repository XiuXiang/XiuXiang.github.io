About My Blog

Posted 2019-12-10updated 2019-12-15

Welcome to my blog. This blog is based on Hexo. This blog is hosted on github. The domain name is bought in aliyun.

Quick Start Hexo

Check documentation for more info(Writing, Server, Generating, ). If you get any problems when using Hexo, you can find the answer in troubleshooting or you can ask Hexo on GitHub.

The Tutorial I Refered

The first tutorial I found is weiyang’s Blog, which is particular for me. After I followed the above tutorial and have a general understanding, I find a Blog that suited my aesthetic value. So I download the bolg’s theme and used in my Blog. The theme is reported, the origin author is Hux ( his blog ), and the boilerplate is reported on his github. But the origin theme is based on Jkyll, which is similar to Hexo. I want to use Hux’s theme, so I tried to install Jkyll. Well, there is something wrong when I used Jkyll. I just want to have a Blog to record slam technology, but I don’t want to waste too much time on front end technology. So I go back to use the reported Hexo theme.

There is some tutorial to use this theme. Tutorial by Hux. Tutorial by huweihuang

I found that this theme doesn’t support latex equation. That’s a big problem for me. so I search “github blog hexo latex”, and I get the key word : “mathjax”. so I search “github hexo mathjax”, I found several tutorial, and this blog solved my problem.
